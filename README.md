# goit-js-hw-06

### Про модуль

## Цей модуль допомагає закріпити практичні навички роботи з ключовим словом this та об’єктно-орієнтованим програмуванням в JavaScript:

    •	Допомагає розуміти, що таке ключове слово this у контексті окремої функції.
    •	Допомагає визначати значення this у різних контекстах:
          - у глобальній області видимості,
          - у методі об’єкта,
          - у стрілочних функціях,
          - у callback-функціях.
    •	Допомагає знати і застосовувати методи call, apply і bind для керування контекстом.
    •	Допомагає розуміти сутність ООП, поняття класу, екземпляра та інтерфейсу.
    •	Допомагає знати, що таке прототипне наслідування та специфіку його використання.
    •	Допомагає використовувати прототипне наслідування та класи для створення однотипних об’єктів з однаковим набором властивостей, але різними значеннями.
