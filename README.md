# goit-js-hw-06

### Про модуль

## У цьому модулі я закріплюю практичні навички роботи з ключовим словом this та об’єктно-орієнтованим програмуванням в JavaScript:

    •	Розумію, що таке ключове слово this у контексті окремої функції.
    •	Вмію визначати значення this у різних контекстах:
           у глобальній області видимості,
           у методі об’єкта,
           у стрілочних функціях,
           у callback-функціях.
    •	Знаю і вмію застосовувати методи call, apply і bind для керування контекстом.
    •	Розумію сутність ООП, поняття класу, екземпляра та інтерфейсу.
    •	Знаю, що таке прототипне наслідування та специфіку його використання.
    •	Вмію використовувати прототипне наслідування та класи для створення однотипних об’єктів з однаковим набором властивостей, але різними значеннями.
